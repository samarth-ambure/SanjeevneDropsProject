#Sanjeevne Drops - Blood Bank Management System
<br>
Sanjeevne Drops is a full-stack web application designed to streamline blood bank operations, including donor registration,
<br>
inventory management, and hospital requests. The system provides a secure platform for healthcare providers to manage life-saving resources efficiently.
<br>
Features
<br>
User Authentication: Secure login and registration using JWT (JSON Web Tokens).
<br>
Donor Management: Add, update, and track donor details and history.
<br>
Inventory Tracking: Real-time monitoring of blood units available by group (A+, B-, O+, etc.).
<br>
Hospital Requests: System for hospitals to place and track urgent blood requirements.
<br>
Dashboard: Intuitive UI for administrators to view key metrics and inventory status.
